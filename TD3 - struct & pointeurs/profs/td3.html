Préambule

Le but de ce TD est de se familiariser avec la syntaxe du C, et de pouvoir compiler quelques programmes simples, autour des structures de données et de l'utilisation des fichiers.

Exercices

La suite d'exercices proposée permet de revoir l'ensemble des notions vues en cours. Elles ne seront pas toutes corrigées lors de la séance. Pour certains exercices, un ensemble de tests unitaires peut être fourni afin de vous permettre de tester les programmes.

Structures de données

  1. Définir une structure de données représentant les coordonnées d'un point dans un espace 2D. Les coordonnées sont réelles.

On considère pour les exercices 2 à 10, la structure de données définie dans l'exercice 1.

  1. Écrire une fonction qui affiche les coordonnées d'un point sous la forme : (x, y).
  2. Écrire une fonction qui applique une translation à un point. La fonction retournera la nouvelle valeur du point.
  3. Définir deux structures de données représentant un cercle et un rectangle. Le cercle contient un point et un rayon, réel. Le rectangle contient deux points, qui sont les deux points de l'une de ses diagonales.

On considère pour les exercices 5 à 10, les structures de données définies dans l'exercice 4.

  1. Écrire une fonction qui calcule le périmètre et l'aire d'un cercle.

  2. Écrire une fonction qui calcule le périmètre et l'aire d'un rectangle.

  3. Écrire une fonction qui applique une translation à un cercle.

  4. Écrire une fonction qui applique une translation à un rectangle.

  5. Écrire une fonction qui indique si un point est présent dans un cercle.

  6. Écrire une fonction qui indique si un point est présent dans un rectangle.

  7. Définir une structure de donnée qui permet de représenter un temps en heures, minutes et secondes.

On considère pour les exercices 12 à 16, la structure de données définie dans l'exercice 11.

  1. Écrire une fonction qui affiche le temps sous la forme : 12h 34m 56s.
  2. Écrire une fonction qui convertit un temps h/m/s en secondes.
  3. Écrire une fonction qui convertit un temps secondes en h/m/s.
  4. Écrire une fonction qui additionne deux temps.
  5. Écrire une fonction qui soustrait deux temps.

Fichiers

  1. Écrire une fonction qui écrit une chaîne de caractères dans un fichier.
  2. Écrire une fonction qui lit une chaîne de caractères à partir d'un fichier.

On considère pour les exercices 19 et 20, la structure de données définie dans l'exercice 11.

  1. Écrire une fonction qui écrit un temps dans un fichier.
  2. Écrire une fonction qui lit un temps à partir d'un fichier.
